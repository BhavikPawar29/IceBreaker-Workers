# IceBreaker Workers

Breaking Ice is a fast conversation assistant for awkward real-life moments. It helps people get one natural line to say without scrolling through a huge feed.

The main app focuses on:

- `live` for quick conversation help
- `explore` for browsing ideas
- `top picks` for standout lines
- `profile` for personal contributions

This repo is the backend Worker repo for Breaking Ice.

It owns the Cloudflare Workers code that will gradually move selected app features off direct client access and into small server-side APIs.
